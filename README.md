# Estudio Juridico
Este proyecto consiste en una web estática para una abogada, desarrollada con HTML y CSS. La web proporciona información profesional y detalla los servicios legales ofrecidos. Está alojada en GitHub Pages para facilitar su acceso y visualización. Puedes visitarla en: [Cyntia Guzmán Abogada.](https://cyntiaguzmanabogada.github.io/CyntiaGuzman/)
